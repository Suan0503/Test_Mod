# LineBot_New 專案

## 專案說明
這是一個模組化、UI美觀的 LINE BOT 專案範本，採用 Python + Flask + line-bot-sdk。

## 目錄結構
- app.py                # 主入口
- config.py             # 設定檔
- requirements.txt      # 依賴套件
- handlers/             # LINE事件/訊息處理
- models/               # 資料庫模型
- services/             # 業務邏輯
- utils/                # 工具函式
- templates/            # 訊息/網頁模板
- static/               # 靜態檔案（CSS/圖片）

## 快速開始
1. 安裝依賴：pip install -r requirements.txt
2. 設定 config.py
3. 執行：python app.py

## 主要技術
- Flask
- line-bot-sdk
- SQLAlchemy
- Bootstrap 5
